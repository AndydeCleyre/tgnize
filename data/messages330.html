How to set GPG with Git in Windows / get a verified badge next to your commits in GitHub and GitLab

1) Install GnuPG. Download it from https://gnupg.org (direct link https://www.gnupg.org/ftp/gcrypt/binary/gnupg-w32-2.2.17_20190709.exe 4.08MB) or full-featured version from https://gpg4win.org (direct link https://files.gpg4win.org/gpg4win-3.1.10.exe 27.66MB)

2) Generate a new key
$ gpg --full-generate-key
options:-
kind of key: RSA and RSA
key size: 4096
valid for: 0
name: Your Name
email: Your GitHub or Gitlab registered email
finally give a new password when asked

3) Get your 16 digits key-id
$ gpg --list-secret-keys --keyid-format LONG
sec rsa4096/82EE4B83DD404A98 2019-09-25 [SC]

4) Get your public key to add it in GitHub or GitLab
$ gpg --armor --export 82EE4B83DD404A98
copy from -----BEGIN PGP PUBLIC KEY BLOCK----- to -----END PGP PUBLIC KEY BLOCK-----

5.1) Open GitHub -> go to settings -> click on SSH and GPG keys -> click on New GPG key -> paste -----BEGIN PGP PUBLIC KEY BLOCK----- to -----END PGP PUBLIC KEY BLOCK----- there and click Add GPG key.

5.2) Open GitLab-> go to settings -> click on GPG keys -> paste -----BEGIN PGP PUBLIC KEY BLOCK----- to -----END PGP PUBLIC KEY BLOCK----- there and click Add key.

6) Now tell Git about your new signing key.
$ git config --global user.signingkey 82EE4B83DD404A98
$ git config --global commit.gpgsign true
$ git config --global gpg.program "C:\Program Files (x86)\gnupg\bin\gpg.exe"

If you want to delete your key from your PC use this command
$ gpg --delete-secret-key 82EE4B83DD404A98
